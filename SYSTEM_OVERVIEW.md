# ระบบ Remote Control Android - ภาพรวมสำหรับลูกค้า

## 📱 ระบบคืออะไร?

**ระบบ Remote Control Android** เป็นแพลตฟอร์มที่ให้คุณสามารถควบคุมและใช้งานอุปกรณ์ Android จากระยะไกลผ่านเว็บเบราว์เซอร์ได้แบบเรียลไทม์ เหมาะสำหรับธุรกิจที่ต้องการให้ลูกค้าใช้งานแอปพลิเคชันบน Android โดยไม่ต้องซื้อเครื่องเอง

---

## 🎯 ความสามารถหลักของระบบ

### 1. **ควบคุม Android จากระยะไกล**

- ดูหน้าจอ Android แบบเรียลไทม์ผ่านเว็บเบราว์เซอร์
- คลิก/แตะหน้าจอได้เหมือนใช้งานเครื่องจริง
- ส่งคำสั่งควบคุมได้ทันที (ไม่มีความล่าช้า)

### 2. **จัดการผู้ใช้งาน (User Management)**

- สร้างบัญชีผู้ใช้ได้ไม่จำกัด
- จัดการเวลาใช้งาน (เติมเวลา 1 ชั่วโมง, 1 วัน, 1 สัปดาห์, 1 เดือน, 1 ปี)
- ติดตามการใช้งานแบบเรียลไทม์
- ระบบหยุด/เริ่มต่อ session ได้ (เวลาไม่เสียเปล่า)

### 3. **จัดการอุปกรณ์ (Device Management)**

- ลงทะเบียนอุปกรณ์ Android ได้หลายเครื่อง
- ดูสถานะเครื่องแบบเรียลไทม์ (พร้อมใช้งาน / กำลังใช้งาน / ออฟไลน์)
- ย้าย session ระหว่างเครื่องได้ (เมื่อเครื่องหนึ่งมีปัญหา)
- ระบบจะบันทึกประวัติการย้ายเครื่องอัตโนมัติ

### 4. **จัดการ Session (Session Management)**

- สร้าง session ใหม่เมื่อลูกค้าจ่ายเงิน
- ติดตามเวลาที่เหลือแบบเรียลไทม์
- หยุด session ชั่วคราวได้ (เวลาไม่นับต่อ)
- เริ่ม session ต่อได้ (เวลาเริ่มนับต่อ)
- ย้าย session ไปเครื่องอื่นได้ (พร้อมบันทึกเหตุผล)
- ยกเลิก session ได้

### 5. **Dashboard สำหรับ Admin**

- ดูภาพรวมระบบแบบเรียลไทม์
- สรุปจำนวนเครื่อง (กำลังใช้งาน / พร้อมใช้งาน / ผิดพลาด / ซ่อมบำรุง)
- ดูรายละเอียดทุกเครื่องในรูปแบบกริด
- ค้นหาและกรองเครื่องตามสถานะ
- จัดการผู้ใช้และอุปกรณ์ทั้งหมด

---

## 🏗️ สถาปัตยกรรมระบบ

### **3 ส่วนหลัก:**

#### 1. **Backend Server (NestJS)**

- รับ-ส่งข้อมูลระหว่าง Web และ Android
- จัดการฐานข้อมูล (Users, Devices, Sessions)
- จัดการการเชื่อมต่อแบบเรียลไทม์ (WebSocket)
- ระบบความปลอดภัย (Authentication & Authorization)

#### 2. **Web Dashboard (Next.js)**

- **Admin Panel** - สำหรับผู้ดูแลระบบ
  - จัดการผู้ใช้, อุปกรณ์, และ Session
  - ดูภาพรวมระบบแบบเรียลไทม์
- **User Panel** - สำหรับลูกค้า
  - เลือกเครื่อง Android ที่ต้องการ
  - ควบคุมเครื่องผ่านเว็บเบราว์เซอร์

#### 3. **Android Agent (Kotlin)**

- ติดตั้งบนเครื่อง Android จริง
- ส่งภาพหน้าจอไปยัง Server แบบเรียลไทม์
- รับคำสั่งควบคุมจาก Web และทำการคลิก/แตะหน้าจอ

---

## 🔐 ระบบความปลอดภัย

- **Authentication** - ระบบล็อกอินด้วย JWT Token
- **Authorization** - แยกสิทธิ์ Admin และ User
- **Password Encryption** - รหัสผ่านถูกเข้ารหัสด้วย bcrypt
- **Session Management** - จัดการ session อย่างปลอดภัย

---

## 📊 ฐานข้อมูล

ระบบใช้ **MongoDB** เก็บข้อมูล:

- **Users** - ข้อมูลผู้ใช้, เวลาใช้งาน, สถานะ
- **Devices** - ข้อมูลเครื่อง Android, สถานะ, ข้อมูลเทคนิค
- **Sessions** - ข้อมูล session, เวลาที่เหลือ, สถานะ
- **Session Move Logs** - ประวัติการย้าย session

---

## 🚀 การใช้งาน

### **สำหรับ Admin:**

1. ล็อกอินเข้าสู่ระบบ Admin Panel
2. สร้างผู้ใช้ใหม่และเติมเวลาให้
3. เชื่อมผู้ใช้กับเครื่อง Android
4. สร้าง Session เมื่อลูกค้าจ่ายเงิน
5. ดูภาพรวมระบบและจัดการทุกอย่างผ่าน Dashboard

### **สำหรับ User (ลูกค้า):**

1. ล็อกอินเข้าสู่ระบบ User Panel
2. เลือกเครื่อง Android ที่พร้อมใช้งาน
3. เริ่ม Session และควบคุมเครื่องผ่านเว็บเบราว์เซอร์
4. ดูเวลาที่เหลือแบบเรียลไทม์

---

## 💡 จุดเด่นของระบบ

✅ **Real-time** - ภาพหน้าจอและคำสั่งควบคุมแบบเรียลไทม์  
✅ **Scalable** - รองรับหลายเครื่องและหลายผู้ใช้พร้อมกัน  
✅ **Flexible** - ย้าย session ระหว่างเครื่องได้  
✅ **User-friendly** - ใช้งานง่ายทั้งฝั่ง Admin และ User  
✅ **Secure** - ระบบความปลอดภัยครบถ้วน  
✅ **Trackable** - ติดตามการใช้งานและประวัติครบถ้วน
