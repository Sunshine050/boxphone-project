## สำหรับผู้ใช้ทั่วไป (User)

### เข้าใช้งานครั้งแรก

1. เปิดลิงก์ที่ Admin ให้มา
2. กรอก **ชื่อผู้ใช้** และ **รหัสผ่าน** ที่ Admin แจกให้
3. กด **เข้าสู่ระบบ**

> ถ้าลืมรหัสผ่าน ต้องแจ้ง Admin ให้รีเซ็ตให้ เปลี่ยนเองไม่ได้

---

### หน้าหลัก — CloudPhone Devices

หลังเข้าสู่ระบบจะเห็นรายการมือถือที่ Admin มอบหมายให้คุณ แต่ละเครื่องจะแสดง:

- **ชื่อมือถือ**
- **เวลาคงเหลือ** — ตัวเลขสีฟ้า (นาที:วินาที) ที่นับถอยลงเรื่อยๆ
- **สถานะ** — ว่าใช้งานได้หรือไม่

**วิธีควบคุมมือถือ:**
1. คลิกที่เครื่องที่ต้องการ
2. รอภาพหน้าจอมือถือโหลดขึ้นมา (อาจใช้เวลา 2-5 วินาที)
3. ควบคุมได้ทันที

---

### วิธีใช้หน้าจอควบคุม

| ทำอะไร | วิธีทำ |
|--------|--------|
| **กดสิ่งที่ต้องการ** | คลิกบนหน้าจอมือถือ |
| **เลื่อนหน้าจอขึ้น/ลง** | คลิกค้างแล้วลาก |
| **กดปุ่ม Back** | กดปุ่ม **Back** ด้านล่างหน้าจอ |
| **กลับหน้าหลักมือถือ** | กดปุ่ม **Home** ด้านล่างหน้าจอ |
| **ดูแอปที่เปิดอยู่** | กดปุ่ม **Recents** ด้านล่างหน้าจอ |
| **ขยายหน้าจอเต็ม** | กดไอคอน **ขยายจอ** มุมขวาบน |

---

### เวลาคงเหลือและการหมดเวลา

- ตัวเลข **สีฟ้า** = ยังใช้งานได้ปกติ
- ตัวเลข **สีแดง** = เวลาเหลือน้อยแล้ว
- ขึ้นว่า **"หมดเวลาใช้งาน"** = หมดเวลาแล้ว ใช้งานต่อไม่ได้

> เมื่อหมดเวลา ต้องแจ้ง Admin เพื่อขอเพิ่มเวลา

---

### การแจ้งเตือน

มีกระดิ่งแจ้งเตือนอยู่มุมขวาบน — ระบบจะส่งแจ้งเตือนเมื่อ:
- Admin เพิ่มเวลาให้
- เวลาถูกลด
- มีการเปลี่ยนแปลงจาก Admin

### ออกจากระบบ

กดปุ่ม **"ออกจากระบบ"** มุมขวาบน → กด **"ออกจากระบบ"** ในป๊อปอัปเพื่อยืนยัน

## สำหรับผู้ดูแลระบบ (Admin)

### เข้าสู่ระบบ Admin

1. เปิดลิงก์ Admin Panel ที่ได้รับ
2. กรอก Username และ Password ของ Admin
3. กด **เข้าสู่ระบบ**

---

### หน้า Overview — ภาพรวมระบบ

หน้าแรกหลัง Login จะเห็นภาพรวมของมือถือทั้งหมด พร้อมสถานะปัจจุบัน

**สิ่งที่ทำได้:**
- ดูมือถือทุกเครื่องและสถานะ
- กด **"Sync จากเสี่ยวเหว๋ย"** เพื่อดึงรายการมือถือใหม่เข้าระบบอัตโนมัติ (ต้องเปิดโปรแกรม Xiaowei ไว้ก่อน)
- กดที่มือถือเพื่อ Assign ให้ผู้ใช้

---

### หน้า Devices — จัดการมือถือ

**เพิ่มมือถือใหม่ด้วยตัวเอง:**
1. กด **"เพิ่มอุปกรณ์ใหม่"**
2. กรอกชื่อ, Serial Number, รุ่น
3. กด Save

**เปลี่ยนสถานะมือถือ:**
- เลือกมือถือ → เลือก Mark Status → เลือกสถานะที่ต้องการ

| สถานะ | ความหมาย |
|-------|----------|
| AVAILABLE | ว่าง พร้อมให้ User ใช้ |
| INUSE | มี User ใช้งานอยู่ |
| OFFLINE | ไม่ได้เชื่อมต่อกับระบบ |
| QUARANTINE | หยุดใช้ชั่วคราว (เช่น ระหว่าง Pause session) |
| UNDER_REPAIR | อยู่ระหว่างซ่อม |
| DAMAGED | เสียหาย ใช้งานไม่ได้ |

---

### หน้า Users — จัดการผู้ใช้

**สร้างผู้ใช้ใหม่:**
1. กด **"สร้างผู้ใช้"**
2. กรอกชื่อ, ชื่อผู้ใช้, รหัสผ่าน
3. กด Save

**มอบมือถือและเวลาให้ผู้ใช้:**
1. คลิกที่ผู้ใช้ที่ต้องการ
2. เลือก **Assign Devices**
3. เลือกมือถือและกำหนดจำนวนเวลา (เป็นวินาที)
4. กด Confirm

> ตัวอย่างการคิดเวลา: 1 ชั่วโมง = 3,600 วินาที, 8 ชั่วโมง = 28,800 วินาที

**เพิ่มเวลาให้ทุกคนพร้อมกัน:**
- กด **"เพิ่มเวลาให้คนที่กำลังใช้งาน"** → ใส่วินาทีที่ต้องการเพิ่ม → กด Confirm

---

### Session — การใช้งานของ User

Session คือ "ช่วงเวลาที่ User กำลังใช้มือถืออยู่" เวลานับถอยลงเรื่อยๆ จนหมด

**สถานะ Session:**

| สถานะ | ความหมาย |
|-------|----------|
| ACTIVE | กำลังใช้งาน เวลากำลังเดิน |
| PAUSED | หยุดพักชั่วคราว เวลาหยุดนิ่ง |
| DISCONNECTED | มือถือหลุดจากระบบ |
| COMPLETED | หมดเวลาแล้ว |
| CANCELLED | Admin ยกเลิก |

**Pause / Resume:**
เลือก Session → กด **Pause** หรือ **Resume**

**ย้าย Session ไปมือถือเครื่องอื่น:**
เลือก Session → กด **Move** → เลือกมือถือปลายทาง
(เวลาที่เหลืออยู่จะถูกย้ายไปพร้อมกัน)

**ยกเลิก Session:**
เลือก Session → กด **Cancel**

---

### หน้า Activity Logs — ประวัติการใช้งาน

ดูทุกการกระทำในระบบ เช่น ใครสร้างอะไร ใครเพิ่มเวลาให้ใคร

**ล้าง Log:**
กด **"เคลียร์ log"** → กด **"ยืนยันเคลียร์"**
- ถ้าตั้งค่า Google Sheets ไว้: ระบบจะส่งออกไป Sheet ก่อนแล้วค่อยลบ
- ถ้าไม่ได้ตั้ง: ลบออกเลยโดยไม่มีสำรอง

> ระบบจะล้าง Log เก่าอัตโนมัติทุกวันที่ 1 ของเดือน

## เมื่อเกิดปัญหา — แก้ยังไง

---

### เข้าสู่ระบบไม่ได้ / รหัสผ่านผิด

**สิ่งที่ต้องทำ:**
1. ตรวจสอบว่าพิมพ์ชื่อผู้ใช้และรหัสผ่านถูกต้อง (ตัวพิมพ์ใหญ่เล็กมีผล)
2. ถ้ายังไม่ได้ — แจ้ง **Admin** เพื่อรีเซ็ตรหัสผ่าน
3. ถ้าเป็น Admin เอง — ตรวจสอบว่า Backend ทำงานอยู่ และค่า `ADMIN_USERNAME` / `ADMIN_PASSWORD` ใน `.env` ถูกต้อง แล้ว restart Backend ใหม่

---

### เข้าไปแล้ว ไม่เห็นมือถือเลย

**สิ่งที่ต้องทำ:**

สำหรับ **User:**
- แจ้ง Admin ว่ายังไม่ได้รับมอบหมายมือถือ
- Admin ต้อง Assign Device ให้ก่อน

สำหรับ **Admin:**
- ตรวจสอบว่ามีมือถือในระบบไหม (หน้า Devices)
- ถ้าไม่มี ให้กด "Sync จากเสี่ยวเหว๋ย" หรือเพิ่มมือถือด้วยตัวเอง
- ถ้ามีแล้วแต่ User ยังไม่เห็น ให้ไปหน้า Users → Assign Devices ให้ User นั้น

---

### ภาพหน้าจอมือถือไม่ขึ้น / โหลดค้าง

**สิ่งที่ต้องทำตามลำดับ:**

1. **Refresh หน้าเว็บ** (กด F5 หรือ Ctrl+R)
2. ตรวจสอบว่ามือถือเครื่องนั้น **ยังเปิดอยู่และมีสัญญาณ**
3. ถ้ายังไม่ได้ — แจ้ง Admin

สำหรับ **Admin:**
- ตรวจสอบสถานะมือถือในหน้า Devices ว่าเป็น ONLINE หรือ OFFLINE
- ถ้า OFFLINE แปลว่ามือถือหลุดจากระบบ ต้องตรวจสอบการเชื่อมต่อฝั่งเครื่อง Server
- ตรวจสอบว่าโปรแกรม Xiaowei ยังรันอยู่

---

### ควบคุมมือถือไม่ได้ / กดแล้วไม่ตอบสนอง

**สิ่งที่ต้องทำตามลำดับ:**

1. รอสักครู่แล้วลองใหม่ — บางครั้งเน็ตช้าทำให้หน่วง
2. Refresh หน้าเว็บ
3. ออกจากระบบแล้วเข้าใหม่
4. ถ้ายังไม่ได้ — แจ้ง Admin

สำหรับ **Admin:**
- ตรวจสอบว่า Session ของ User นั้นยัง ACTIVE อยู่ไหม
- ลองกด Pause แล้ว Resume ใหม่
- ถ้ามือถือเป็น OFFLINE ให้ตรวจสอบการเชื่อมต่อ USB หรือ Agent บนมือถือ

---

### เวลาหมดโดยที่ยังไม่ได้ใช้ หรือหมดเร็วกว่าที่ควร

**สิ่งที่ต้องทำ:**
- แจ้ง Admin เพื่อตรวจสอบ Session และเพิ่มเวลาให้

สำหรับ **Admin:**
- ไปหน้า Users → เลือก User → ดูเวลาคงเหลือ
- กด "เพิ่มเวลาให้คนที่กำลังใช้งาน" หรือ Assign เวลาเพิ่มให้ User นั้น
- ถ้าปัญหาเกิดซ้ำบ่อย ให้ตรวจสอบว่านาฬิกาของ Server ตรงกับเวลาจริง

---

### กดปุ่มใดๆ แล้วขึ้น "กรุณาเข้าสู่ระบบใหม่" หรือถูก Redirect ไปหน้า Login

**สิ่งที่ต้องทำ:**
- เข้าสู่ระบบใหม่ — Token หมดอายุ (ทุก 24 ชั่วโมง)
- ถ้าเกิดบ่อยผิดปกติ แจ้ง Admin

---

### ระบบแจ้งเตือนไม่มา

**สิ่งที่ต้องทำ:**
1. ตรวจสอบไอคอนกระดิ่งมุมขวาบน — กดเพื่อดูแจ้งเตือนที่ค้างอยู่
2. ลอง Refresh หน้าเว็บ
3. ถ้ายังไม่มา — แจ้ง Admin

---

### มือถือขึ้นสถานะ OFFLINE ทั้งที่เพิ่งใช้อยู่ได้

**สิ่งที่ต้องทำ (Admin):**
1. ตรวจสอบว่าสาย USB ที่เสียบมือถืออยู่กับ Server ยังเชื่อมต่ออยู่
2. ตรวจสอบว่าโปรแกรม Xiaowei หรือ USB Bridge ยังรันอยู่
3. ถ้า Agent ที่ติดตั้งในมือถือ — เปิดแอป BoxPhone Agent ในมือถือและกด Connect ใหม่
4. ถ้ายังไม่ได้ — Restart Service แล้วรอ 30 วินาที

---

### ล็อกอิน Admin แล้วเข้าไม่ได้เลย ทั้งที่รหัสถูก

**สิ่งที่ต้องทำ (Admin / ทีมเทคนิค):**
- อาจเป็น Backend ไม่ได้รัน — ให้ทีมเทคนิค Restart Backend
- ถ้าเพิ่งแก้ไฟล์ตั้งค่า ต้อง Restart Backend ใหม่ทุกครั้ง
- ตรวจสอบกับทีมเทคนิคว่า Database (MongoDB) กำลังทำงานอยู่

---

### สิ่งที่ทำเองไม่ได้ — ต้องแจ้งทีมเทคนิคเสมอ

- ระบบล่มทั้งหมด เข้าไม่ได้เลย
- มือถือ OFFLINE หลายเครื่องพร้อมกัน
- ต้องการเพิ่ม/เปลี่ยนแปลงการตั้งค่าระบบ
- ต้องการ Export Log ไป Google Sheets (ต้องตั้งค่าก่อน)
- ต้องการ Backup ข้อมูล
