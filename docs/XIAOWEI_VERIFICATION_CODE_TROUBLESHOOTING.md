# คู่มือแก้ปัญหา: รหัสยืนยันไม่ถูกต้อง

## ❌ ปัญหา: "无效验证码" (Invalid verification code)

เมื่อพยายาม register/login ในเสี่ยวเหว๋ย:
- **ใส่รหัสยืนยันที่ส่งมาในอีเมลแล้ว** แต่ยังบอกว่า "รหัสไม่ถูกต้อง"
- **Error message:** "无效验证码" (Invalid verification code)

---

## 🔍 สาเหตุที่เป็นไปได้

### 1. ⏰ รหัสหมดอายุ (Expired)

- **รหัสยืนยันมีอายุจำกัด** (มักจะ 60-300 วินาที)
- **ถ้า countdown timer หมด** → รหัสใช้ไม่ได้แล้ว
- **วิธีแก้:** ขอรหัสใหม่

### 2. 🔄 รหัสถูกใช้ไปแล้ว

- **ถ้าส่งรหัสใหม่** → รหัสเก่าจะใช้ไม่ได้
- **รหัสยืนยันใช้ได้แค่ครั้งเดียว**
- **วิธีแก้:** ใช้รหัสล่าสุดที่ส่งมา

### 3. 📧 รหัสไม่ตรง (มี space หรือตัวอักษรพิเศษ)

- **อาจมี space หรือตัวอักษรพิเศษ** ที่ copy มา
- **หรือตัวอักษรที่คล้ายกัน** (เช่น 0 vs O, 1 vs I)
- **วิธีแก้:** ตรวจสอบและลบ space/ตัวอักษรพิเศษ

### 4. ⏱️ Time Sync Issue

- **เวลาระหว่าง client และ server ไม่ตรงกัน**
- **ทำให้รหัสหมดอายุเร็วกว่าที่ควร**
- **วิธีแก้:** ตรวจสอบเวลาในระบบ

### 5. 📨 รหัสถูกส่งไปหลายครั้ง

- **ถ้าส่งรหัสหลายครั้ง** → รหัสเก่าจะใช้ไม่ได้
- **ต้องใช้รหัสล่าสุดที่ส่งมา**
- **วิธีแก้:** ใช้รหัสจากอีเมลล่าสุด

---

## ✅ วิธีแก้ไข

### วิธีที่ 1: ขอรหัสใหม่ (แนะนำ)

1. **กดปุ่ม "Send Code" หรือ "重新发送" (Resend)**
2. **รอรับอีเมลใหม่**
3. **ใช้รหัสล่าสุดที่ส่งมา** (ไม่ใช้รหัสเก่า)
4. **ใส่รหัสทันทีหลังจากได้รับ** (ก่อนหมดอายุ)

### วิธีที่ 2: ตรวจสอบรหัส

1. **ตรวจสอบว่าไม่มี space หรือตัวอักษรพิเศษ**
   - ลบ space หน้า-หลัง
   - ตรวจสอบตัวอักษรที่คล้ายกัน (0 vs O, 1 vs I)

2. **Copy รหัสจากอีเมลโดยตรง**
   - ไม่ต้องพิมพ์เอง
   - Copy ทั้งหมดแล้ว paste

3. **ตรวจสอบว่าใช้รหัสล่าสุด**
   - ดู timestamp ในอีเมล
   - ใช้รหัสจากอีเมลล่าสุด

### วิธีที่ 3: ตรวจสอบ Spam/Junk Folder

1. **ตรวจสอบ Spam/Junk Folder** ในอีเมล
2. **รหัสอาจถูกส่งไปหลายครั้ง** → ใช้รหัสล่าสุด
3. **ตรวจสอบว่าไม่ใช่ phishing email**

### วิธีที่ 4: ใช้วิธีอื่นในการ Activate VIP

ถ้ายังไม่ได้:

1. **ลองใช้ HTTP API แทน WebSocket**
   - HTTP API อาจไม่ต้องการ VIP
   - เปิด HTTP API Server ในเสี่ยวเหว๋ย

2. **ติดต่อเสี่ยวเหว๋ย Support**
   - Website: https://www.xiaowei.xin/
   - สอบถามเรื่อง:
     - รหัสยืนยันไม่ถูกต้อง
     - วิธี activate VIP โดยไม่ต้องรหัสยืนยัน
     - วิธีใช้ API โดยไม่ต้อง VIP

## 📝 ขั้นตอนแนะนำ

### Step 1: ขอรหัสใหม่

1. ✅ **กดปุ่ม "Send Code" หรือ "重新发送"**
2. ✅ **รอรับอีเมลใหม่** (ตรวจสอบ Spam/Junk Folder)
3. ✅ **ดู countdown timer** - ต้องใส่รหัสก่อนหมดอายุ

### Step 2: ใช้รหัสล่าสุด

1. ✅ **ใช้รหัสจากอีเมลล่าสุด** (ไม่ใช้รหัสเก่า)
2. ✅ **Copy รหัสจากอีเมลโดยตรง** (ไม่ต้องพิมพ์เอง)
3. ✅ **ตรวจสอบว่าไม่มี space หรือตัวอักษรพิเศษ**

### Step 3: ใส่รหัสทันที

1. ✅ **ใส่รหัสทันทีหลังจากได้รับ** (ก่อนหมดอายุ)
2. ✅ **ไม่รอให้ countdown timer หมด**
3. ✅ **ถ้า countdown หมด → ขอรหัสใหม่**

## 💡 Tips

- **รหัสยืนยันมีอายุจำกัด** - ต้องใส่ทันทีหลังจากได้รับ
- **รหัสใช้ได้แค่ครั้งเดียว** - ถ้าส่งรหัสใหม่ รหัสเก่าจะใช้ไม่ได้
- **ตรวจสอบ Spam/Junk Folder** - รหัสอาจถูกส่งไปที่ spam
- **ใช้รหัสล่าสุด** - ถ้าส่งรหัสหลายครั้ง ต้องใช้รหัสล่าสุด
- **Copy จากอีเมลโดยตรง** - ไม่ต้องพิมพ์เอง เพื่อหลีกเลี่ยง typo
