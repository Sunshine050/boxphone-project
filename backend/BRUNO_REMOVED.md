# Bruno Collection ถูกลบแล้ว

Bruno collection ถูกลบออกจากโปรเจกต์แล้ว

## ใช้ Postman แทน

โปรดใช้ Postman collection ที่มีอยู่แล้ว:

- **Collection**: `postman_collection.json`
- **Environment** (Optional): `postman_environment.json`
- **Setup Guide**: `POSTMAN_SETUP.md`

## วิธี Import Postman Collection

1. เปิด Postman
2. คลิก **Import** (มุมซ้ายบน)
3. เลือกไฟล์ `postman_collection.json`
4. คลิก **Import**

## วิธี Import Environment (Optional)

1. เปิด Postman
2. คลิก **Import**
3. เลือกไฟล์ `postman_environment.json`
4. คลิก **Import**
5. เลือก Environment ที่ import มา (มุมขวาบน)
